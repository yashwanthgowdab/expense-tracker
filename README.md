# Expense Tracker

A full-stack web application to track personal income and expenses.

## 🌐 Live Demo
👉 https://expense-tracker-l25dgedql-yashwanth-gowda-b-s-projects.vercel.app

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Recharts
- **Backend:** Node.js, Express.js, JWT, Bcrypt
- **Database:** MongoDB Atlas

## Features
- User Signup / Login with JWT Authentication
- Add Income and Expenses with category & date
- Dashboard with charts and balance summary
- Filter transactions by date range
- Export transactions to Excel

## How to Run Locally

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend1
npm install
npm run dev
```

Open `http://localhost:5173` in browser.
